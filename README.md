# Chinmay Agrawal's Portfolio

A visionary's digital space. A multidisciplinary creator's sanctuary. Built with meticulous care and attention to detail.

## Features

- **Smooth Scrolling** - Powered by Lenis for butter-smooth scroll interactions
- **GSAP Animations** - Beautiful entrance animations and scroll-triggered reveals
- **Interactive Contribution Banner** - A canvas-based git-style contribution chart at the top
- **Custom Cursor** - Elegant custom cursor with hover states
- **Abyss Shader Background** - Interactive WebGL shader for the testimonials section
- **Spotify Integration** - Display your favorite music directly on the site
- **Responsive Design** - Fully responsive across all devices
- **Dark Aesthetic** - Premium dark theme with gold accents

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, flexbox, grid, animations
- **JavaScript (Vanilla)** - No frameworks, pure JS
- **GSAP** - Professional animations
- **Lenis** - Smooth scrolling
- **WebGL** - Interactive shader backgrounds

## Structure

```
├── index.html           # Main HTML file
```

## Getting Started

### Local Development

Run a local server:

```bash
python3 -m http.server 3000
```

Then visit `http://localhost:3000`

### Deployment

This site is optimized for deployment on Vercel, Netlify, or GitHub Pages.
